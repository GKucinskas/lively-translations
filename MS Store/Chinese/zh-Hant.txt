描述：
本軟體為免費的開源應用程式，用於將 GIF、影片和網頁設定為您的桌面背景。
由 Chromium 瀏覽器和 MPV 播放器提供支援，故您的桌布將支援最新的網路技術和影片標準。

這是一個開源專案，歡迎社群的參與。欲加入請訪問
https://github.com/rocksdanister/lively (請使用英文)

特徵：
在執行遊戲/進行工作時，暫停播放桌布（0% 資源佔用）。
支援任何縱橫比和解析度的多顯示器。
由 mpv 播放器提供支援的硬體加速影片播放，可將任何影片設定為桌布。
由 Chromium Embedded Framework (CEF) 提供支援，將任何網站設定為桌布。
可使用 Lively Audio API 製作對系統音訊做出回應的桌布。
可使用 Lively System API 製作展示系統資訊的小工具。
支援線上影片串流和 Shadertoy.com 連結。
完全開源和免費，沒有付費牆阻擋功能使用。
支援深色和淺色主題。

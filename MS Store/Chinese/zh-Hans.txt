描述：
免费的开源应用程序，用于将 GIF、视频和网页设置为您的桌面壁纸。
由 Chromium 浏览器和 MPV 播放器提供支持，您的壁纸支持最新的网络技术和视频标准。

这是一个开源项目，欢迎社区参与。参加请访问
https://github.com/rocksdanister/lively

特征：
运行游戏/工作时，壁纸播放暂停（0% 资源使用）。
支持任何纵横比和分辨率的多个显示器。
由 mpv 播放器提供支持的硬件加速视频播放，将任何视频设置为墙纸。
由 Chromium Embedded Framework (CEF) 提供支持，将任何网站设置为墙纸。
使用 Lively Audio API 制作对系统音频做出反应的壁纸。
使用 Lively System API 制作显示系统信息的小部件。
支持在线视频流和 Shadertoy.com 链接。
完全开源和免费，付费专区没有任何功能。
支持深色和浅色主题。
